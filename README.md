# Library Management System

## Опис проекту
Система управління бібліотекою на JavaScript. Реалізовано за допомогою функціональних об'єктів та замикань.

## Функціонал
- **Book Object**: зберігає дані про книгу, має методи `borrow()`, `returnBook()` та геттер `available`.
- **Library Manager**: керує масивом книг, підтримує пошук та фільтрацію.

## Технічні особливості
- Використання **ES6 Modules** (import/export).
- **Closures**: приватні змінні для статусу книги та списку книг.
- **Immutability**: використання spread-оператора для оновлення даних.
- **Unit Testing**: покриття коду тестами за допомогою Jest.

## Як запустити
1. `npm install` — встановлення залежностей.
2. `npm start` — запуск демонстрації.
3. `npm test` — запуск автоматичних тестів.
